# What is NgCoreApp?
The NgCoreApp is a Small Application to learn how to CRUD (Create, Read, Update, Delete) Operation performed Using Asp.Net Core Web Api With Angular.

## Technologies Used
•	.Net 8.0
•	Entity Framework Core 8.0.0
•	SQLite Database
•	Angular 12
•	Clean Architecture
•	MediatR With CQRS
•	SQLite Database
•	Angular 12
