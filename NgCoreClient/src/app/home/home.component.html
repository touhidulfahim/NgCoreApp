NGCOREAPP

The NgCoreApp is a Small Application to learn how to CRUD (Create, Read, Update, Delete) Operation performed Using Asp.Net Core Web Api With Angular.

  • Technologies Used (Back-End)
    • Clean Architecture
    • ASP.NET Core Web API
    • ASP.NET Core 8
    • Entity Framework Core 8
    • MediatR
    • CQRS
    • FluentValidation
    • Repository Pattern, Factory Pattern, Unit Of Work Pattern, Dependency Injection
    • SQLite Database
  • Technologies Used (Front-End)
    • Angular 12
    • Bootstrap 5.3.2
    • DatePipe For Formatted-Date Purpose
  • Key Points to Consider:
    • These projects have been designed with a focus on clean code principles.
